# ProjectDELTA
The goal of ProjectDELTA is to natively port Deltarune to PlayStation 3.

# Disclaimer
All original games (Chapters 1 and 2) are made by Toby Fox and distributed for free in demo versions. ProjectDELTA is not aimed at porting paid chapters 3 and 4. There is also a disclaimer at the start of the game.

# Info
This is NATIVE port, it doesn't apply to Butterscotch and other utilities.
Currently we have 100% working native port of Chapter 1 by me.
Chapter 2 and building info soon.
You can check progress in my TGC:
https://t.me/tobyporty
