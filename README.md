# ProjectDELTA
The goal of ProjectDELTA is to natively port Deltarune to PlayStation 3 and Vita.

# Disclaimer
All original games (Chapters 1 and 2) are made by Toby Fox and distributed for free in demo versions. ProjectDELTA is not aimed at porting paid chapters 3 and 4. There is also a disclaimer at the start of the game. ProjectDELTA also doesn't distribute ready-made tools for building PKG files and other similar things. By launching build.bat or build.sh, utilities will be downloaded to the user's computer from other repositories on Github that are needed for the build.

# Info
This is NATIVE port, it doesn't apply to Butterscotch and other utilities.
